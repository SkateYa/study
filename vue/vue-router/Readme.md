后端安装koa命令
# koa2    npm i koa-generator -g
# koa2 project-name
# cd project-name
# npm install
# npm i koa2-cors -S   //设置跨域
# npm run dev

启动后端项目： npm run start


# 路由权限
- 1、uid -> 后端 API - 路由权限API
- 2、后端 -> 用户对应路由权限列表 -> 前端 -> JSON
- 3、JSON -> 树形结构化
- 4、树形结构化的数据 -> vue 路由结构
- 5、路由结构动态 -> 静态路由
- 6、树形结构化的数据 -> 菜单组件
